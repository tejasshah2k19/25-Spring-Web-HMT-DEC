Spring Validation => 

step1 : 
		lib / dep / jar => 
					pom.xml -> validation -> 
					spring starter -> validation -> 
					
step2 :
		you have to mark validation inside bean class 
		@NotBlank 
		apply validation using @NotBlank , @NotNull , @Pattern , @Min , @Max
		 
		
step 3:
		you need to check , bean is as per validation or not 
		do we have any validations errors ? 
		@Validated -> validation run 
				and return 
			BindingResult -> error true | false 
		
step 4: 
		using binding result we can check error is present or not 
			result.hasErrors() 
			
			if errors are their send result using model and  return with input jsp 
			else perform your logic 
			
step 5: to print error message on jsp 
			we will use result in EL 
					${result.getFieldError("fieldName").defaultMessage}
